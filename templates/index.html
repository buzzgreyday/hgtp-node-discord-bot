IN-DEPTH REPORTING

Nodebot reports cover essential metrics to assist operators in maintaining the optimal performance, security, and reliability of their nodes. Operators can efficiently monitor node status, connectivity, rewards, versioning, and resource usage through Discord direct messages whenever desired.

MEANINGFUL ALERTS

Operators use services like UptimeRobot to monitor node connections, but these often lack detailed information. Nodebot provides detailed notifications for:

  • Node up, down, or connecting
  • New version available
  • Missed rewards
  • Node forks
  • Low resources
  • Status update every 6 hours if an issue persists
  • Collateral < 250,000 $DAG
  • Unstable LB anti-spam feature
